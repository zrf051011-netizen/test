LOGIN & SECURITY
ENTRY · 07
06 / 登录入口

视觉升级不接管业务逻辑，安全边界仍在服务端

四条不变原则

SERVER FIRST
01不公开测试凭据页面无默认账号，不回显密码。
02统一失败提示避免向用户暴露内部异常信息。
03服务端权限校验前端隐藏不等于权限控制。
04参数化查询DBUtils 绑定参数，降低注入风险。
CSS / Canvas / Shader 仅作为渐进增强层
加载失败时仍保留可用的登录表单
新版登录页 · 液态玻璃
页面负责体验，Servlet 负责身份与权限
